# Contributing

## Standards

- Put intuition before equations.
- Derive results instead of skipping steps.
- Keep implementations readable, typed, and documented.
- Verify important calculations against PyTorch.
- Add or update pytest coverage for source changes.

## Workflow

1. Install dependencies with `python -m pip install -r requirements.txt`.
2. Install the package with `python -m pip install -e .`.
3. Run `python -m pytest`.
4. If you changed notebooks, run them top to bottom before opening a pull request.
